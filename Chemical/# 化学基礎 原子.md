# 化学基礎 原子

## 原子の構造

```
原子
 ┣ 原子核
 ┃ ┣ 陽子
 ┃ ┗ 中性子
 ┗ 電子
```



## 陽子、中性子、電子

|        | 電化 | 質量比 |
| ------ | ---- | ------ |
| 陽子   | +1   | 1      |
| 中性子 | 0    | 1      |
| 電子   | -1   | 1/1840 |



## 原子番号

原子番号は陽子の数



## 質量数

質量数は陽子の数と中性子の数の和



## 同位体

原子番号が等しく、中性子の数が異なる原子
