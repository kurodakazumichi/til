# 化学基礎 物質の三態

## 拡散

物質が自然に広がっていく現象



## 熱運動

物質を構成する分子の不規則な運動のこと

分子は熱を持っていると振動し、拡散の原因にもなる。



| 温度       | 低い       | →    | 高い   |
| ---------- | ---------- | ---- | ------ |
| 熱運動     | おとなしい | →    | 激しい |
| 物質の状態 | 個体       | 液体 | 気体   |



## 絶対温度

分子が熱運動を止める温度を **絶対零度** といい、それを基準にして決めた温度を **絶対温度** という。

そして、その単位を **K** と表し、**ケルビン** と呼ぶ。



絶対温度はセルシウス温度に **273°C** 足した温度になる。

| セルシウス温度 | ケルビン |
| -------------- | -------- |
| 100°C          | 373K     |
| 0°C            | 273K     |
| -273°C         | 0K       |



## 個体・液体・気体＝物質の三態

状態変化

- 融解：個体から液体になること

- 蒸発：液体から気体になること

- 凝縮：気体から液体になること

- 凝固：液体から個体になること

- 昇華：個体から気体、あるいは気体から個体になること



## 物理変化と化学変化

- 物理変化：物質の状態だけ変化(氷→水)

- 化学変化：物質の種類が変化(水素+酸素 → 水)
