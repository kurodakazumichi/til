```
That he ate ten hamburgers / surprised / me.
SVO

自：彼はハンバーガー10個を食べて私を驚かせた。

```



```
How we view things / is affected / by our knowledge.
SVM

自:私たちの物事の見方は私たちの学びに影響した。

```
