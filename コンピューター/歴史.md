# コンピューターの歴史は計算から始まった。

- 農業・牧畜:太陽の動きを予測する計算が求められた -> 天文学
- 建築:土地の面積をどうやって計算するか -> 幾何学
- 経済:あいつに貸した金の利子いくらだっけ -> 会計学
- 軍事:確実に当てるにはどの角度で発射すればいいか -> 物理学

というように、人類はあらゆる分野で計算をするように進化した。

しかし問題点があった

それは**人は間違いを犯す**ということである。

人間が作業をする限り、計算を間違える。
間違いを減らすためにソロバンや計算尺などの道具も発明されたが
あくまでも人間が作業するので間違いはなくならない。

計算を機械に任せられればなぁ

というのがコンピューター発明へ至る基本的な考え方。

# 計算を機械で自動化することを目指す

機械による自動計算を目指し、中世になると**歯車式計算機**が誕生する。

## 1623年 シッカートの計算機

![シッカートの計算機](https://upload.wikimedia.org/wikipedia/commons/thumb/5/5a/Schickardmaschine.jpg/220px-Schickardmaschine.jpg)

## 1642年 パスカリーヌ

![パスカリーヌ](https://upload.wikimedia.org/wikipedia/commons/thumb/8/80/Arts_et_Metiers_Pascaline_dsc03869.jpg/220px-Arts_et_Metiers_Pascaline_dsc03869.jpg)

パスカルが仕事楽になんねーかなーと思って作った計算機。  
足し算と引き算しかできず、耐久性も信頼性も低かったので普及はしなかった模様。

## 1670年代 ライプニッツの計算機

![ライプニッツの計算機](https://upload.wikimedia.org/wikipedia/commons/thumb/9/92/Leibnitzrechenmaschine.jpg/220px-Leibnitzrechenmaschine.jpg)

四則演算が可能。  

「立派な人間が労働者のように計算などという誰でもできることに時間をとられるのは無駄だ。機械が使えたら誰か他の者にやらせるのに」と言ってこれを発明したと言われている。

引き算には「補数」の概念が使われている。
ライプニッツはコンピュータの重要な概念である「2進数」について
初めて数学的な面から言及した人物として知られている。

1と0しかないため、普段使用する「10進数」よりわかりにくい反面
コンピュータの計算回路を単純化できるというメリットがあった。

これは、のちにコンユーターの発展に大きな影響を与えている。



# 歯車式計算機の発展

20世紀に入り、まともな電卓が出てくるまでは姿形を変えながら広い範囲で活躍していた。


# 未完成だがコンピューターの原型が登場


19世紀、産業革命の影響により  
複雑な計算を機械にやらせようとする動きが強まる。

この頃、イギリスのチャールズ・バベッジ(1791~1871)が
現在のコンピュウーたに近い設計思想を持つ計算機を計画した。

## 階差機関

バベッジが最初にとりかかった計算機。
階差表という、対数や三角関数の数表を求めるための計算機。  
度重なる仕様変更により完成はしなかった。

> 数表とは
> 様々な計算の答えを書いた表で、一般的には掛け算九九の表などがある。

## 解析機関

階差機関の次に着手した計算機。  
汎用的な計算を高速かつ自動で実現する事を目的とし、動力には蒸気機関が採用された。

- 現代のアセンブリ言語に近いプログラミング言語を用いようとした。
- プログラムデータはパンチカード(紙に穴をあけたやつ)を採用。
- プログラムと計算機を分離し、プログラムを変える事で様々な計算に対応するという考え方が初めて登場した。
- 解析機関にはCPUやメモリに近い概念も採用されている。

こういった点からバベッジはコンピュータの父と呼ばれている。

しかし当時の技術力では実現が難しかったのか、実際に製造するには至らなかった。

# プログラムが誕生

## 機械とプログラムが分離される利点

バベッジの解析機関ではプログラムをパンチカードに格納した。  
この方法は20世紀になってからも長い間用いられてきた手法である。

プログラム登場以前では、計算のルールは機械の一部に組み込まれていたため
計算可能なものが機械によって限られていた。

プログラムの登場により機械は変えずに、プログラムだけ変える事で別な計算ができるようになった。

## 世界最初のプログラマ

世界最初のプログラマーの名前は

エイダ・ラブレス(享年36才)というイギリスの女性貴族
(と言われている)

趣味は競馬。

# コンピュータ理論の基礎「ブール代数」

19世紀中頃、イギリスのジョージ・ブール(イギリスの数学者)の提唱したブール代数という理論が登場する。
(コンピュータの実用化に重要な理論)

- 真と偽という正しいか、間違いかの２つだけの値を用いる
- AND(かつ)、OR(または)、NOT(~でない)の３つの演算を組み合わせて処理していくもの。

## AND(A,Bどちらも真なら真)

| A | B | A AND B |
| - | - | - |
| 真 | 真 | 真 |
| 真 | 偽 | 偽 |
| 偽 | 真 | 偽 |
| 偽 | 偽 | 偽 |

## OR(A,Bどちらかが真なら真)

| A | B | A OR B |
| - | - | - |
| 真 | 真 | 真 |
| 真 | 偽 | 真 |
| 偽 | 真 | 真 |
| 偽 | 偽 | 偽 |

## NOT(反転)

| A | NOT A |
| - | - |
| 真 | 偽 |
| 偽 | 真 |


# コンピューター理論が揃う

クロード・シャノン(アメリカ)とアラン・チューリング(イギリス)の2人の天才現る

## コンピューターの定義を変えたシャノン

- 1937年、ブール代数の真・偽がスイッチのON・OFFに対応できる事を発見。
- 論理演算のAND回路は直列、OR回路は並列や！
- ブール代数の論理演算はすべてスイッチ回路で実行可能と証明。

これにより「論理を扱う装置」としてコンピューターが生まれることになった。

さらに全ての「情報」を数で数えるものと定義。(情報理論という学問)
全ての情報は数値化できる、そして数値は0と1の組み合わせで表せる事を示した。

## 「計算」と「計算機」の本質を示したチューリング

- チューリングマシンという計算機のモデル(概念)を発表。
- 極めてシンプルな原理、かつあらゆる計算が可能。(神か)
- バレッジが夢見ていた汎用的な計算機が実現可能である事を示した。


# 大規模な計算機の出現(コンピュータが完成に近づく)

第二次世界大戦前後になると、現代のコンピューターに近い構造を持つ計算機が誕生する。

## 1938年 Zuse Z1(ツーゼ　ツェット・アイン)
ドイツ：コンラート・ツーゼ(1910~1995)
- ２進数で演算
- ある程度プログラム可能
- 機械式、電子回路は使われていない
- プログラム内蔵型ではない
- バベッジの考えた解析機関に近い

## 1941年 Zuse Z3(ツーゼ ツェット・ドライ)
ドイツ：コンラート・ツーゼ(1910~1995)
- ２進数で演算
- 全機能をプログラム(テープ)で制御可能
- 演算回路が機械的な要素を持つリレースイッチ
- プログラム内蔵型ではない(テープは外部に格納)
- シャノンの理論を取り入れた

## 1942年 アナソタフ&ベリー・コンピュータ(アメリカ)
ジョン・ヴィンセント・アナソタフ(1903〜1995)  
クリフォード・エドワード・ベリー(1918〜1963)
- ２進数で演算
- 完全な電子回路(真空管)
- コンデンサの記憶装置を搭載
- プログラムの変更には敗戦の変更が必要(プログラム内蔵型ではない)

## 1944年 Colossus(コロッサス)
イギリス：トマス・フラワーズ(1905~1998)
- ２進数で演算
- 完全な電子回路(真空管とサイラトロン)
- ある程度のプログラムが可能(スイッチとプラグの変更で行う)
- プログラム内蔵型ではない


## 1946年 ENIAC(エニアック)
- 開発者：ノイマン、エッカート、モークリー
- 世界初のコンピューターと言われるがノイマン型ではない。
- アメリカ、大砲の弾道計算用に開発された。
- 完成した頃には戦争も終わり結果的に軍事機密西原れず公開された最初のコンピューター。
- 2進数ではなく10進数
- 真空管17468本
- プログラムの変更には回路の変更が必要。
- 全長30m、総面積170m2、重さ30t、消費電力150kW。

# ノイマン型コンピューター

- ハンガリの数学者ノイマンが提唱したもの
- 現代のコンピューターの標準的な構造のことをノイマン型と呼ぶ
- 基本構造は入力装置、記憶装置、制御装置、演算装置、出力装置の５つが備わっていること。


ノイマン型というのはノイマン1人で考えたわけじゃなく
ENIACを開発した腫瘍メンバだった、ジョン・エッカートとジョン・モークリーだったらしい。
2人の考えをまとめて、理論面で裏付けをしたのがノイマンだったようだが、現在は「ノイマン型」という呼び名が定着。
2人のジョンはこれをよく思ってなかったそうな。

## 1948年 The Baby(ザ・ベイビー)
- 世界初のノイマン型コンピューター
- 本名はSSEM(Small Scale Experimental Machine)
- マンチェスター大学のフレディ・ウィリアムスとトム・キルバーンによって作られた。

## 1949年 EDSAC

- イギリスのモーリス・ウィルクスとケンブリッジ大学が共同開発したノイマン型コンピュータ
- EDVAC(後述)の構想に刺激され作成。
- メインメモリに水銀遅延線を使用(なにそれ)
- 超音波のパルスがタンク内の水銀中で遅れて伝播する現象を利用(謎テクノロジー)

## 1951　EDVAC(エドバック)

- ノイマン・エッカート、モークリーらが開発
- ENIACの問題点を解消するために開発
- ノイマン型コンピューターの提唱者たちによる正真正銘のノイマン型コンピューター
- 先述のノイマンとの確執により、エッカートとモークリーが途中脱退し完成は遅れた。

# トランジスタの登場

# 演算素子の発展

- ノイマン型コンピュータの実現によって基本的な構造は完成した。
- 次の問題点は速度と扱える情報の量になった
- 演算速度の向上の鍵はスイッチのオン・オフを行う演算素子を増やすこととスイッチの切り替えを早くすること
- 演算素子は電磁石のリレー式、真空管、トランジスタという流れで発展した
- トランジスタの登場によりコンピュータが爆発的な進化が始まる。

## リレー(1940年以前)
- 安い
- 構造が単純
- スイッチを遠隔操作できる
- 動作が遅い
- ノイズが発生する
- 耐久性が低い

## 真空管(1940年代)
- 高速に動作する
- 耐久性が低い
- 消費電力が大きく発熱する
- 小型化が困難

## トランジスタ(1950年代〜)

- 1948年に、トランジスタが発明された。
- 半導体の特性を利用した演算素子
- 高速に動作すうる
- 消費電力が小さい
- 寿命が長い
- 構造が単純
- 極限まで小型化できる


# マイクロプロセッサの登場

## トランジスタを小さくして沢山集める(集積化)

トランジスタを小さくして集めると

- 演算素子が沢山 → 様々な論理回路を作れる → 処理できるデータ量が増える
- 消費電力が下がる
- 回路の距離が近くなるから速度があがる

トランジスタのサイズ

- 1948年、発明当初は1個が数センチ
- 10年後には数百分の1mmまで小型化

## 集積回路(IC)

電子回路に必要な部品を効率よくまとめて１枚の小さな板に載せたもの。  
ENIAC時代には膨大なスペースが必要だったものが、掌サイズになった。

## マイクロプロセッサの誕生

- 世界初のICはキルビー(アメリカの物理学者)が1958年に開発。

その後もどんどん進化し

- LSI(大規模集積回路)
- 超LSI(超大規模集積回路)

と進化してきた。

その結果

- 最初は演算の役割ごとに複数のICに分かれていた演算処理装置(CPU)が、１枚のチップにまとめられマイクロプロセッサとなった。
- 世界初のマイクロプロセッサ(CPU)は1971年に登場したIntel4004。
- 10マイクロメートルのトランジスタが2300個集まったもの。



## ジャック・キルビー(1923〜2005)
- 世界で最初にICを発明。
- この功績で2000年のノーベル物理学賞を受賞
- テキサス・インスツルメンツ(TI)社はICの特許でぼろ儲け(ジャックのいた会社)

## Intel4004
- 世界初のマイクロプロセッサ
- 開発には日本人もかかわっていた。
- 日本の電卓値下げ、高性能化合戦の産物ともいわれる。

## 集積化のメリット
- 小さいので持ち運べる
- 原料が少なくなり、コストが下がる
- 低消費電力でも高性能
- 発熱が抑えられる
- 計算スピードUP(クロック周波数)できる

## 集積回路の呼称

- IC(Integrated Circuit):集積回路:集積度が1000個以下
- LSI(Large Scale Integration):大規模集積回路：集積度が1000〜10万
- VLSI(Very Large Scale Integration):集積度が10万〜１０００万
- ULSI(Super Large Scale Integration):集積度が1000万個以上

※現在は集積度が数億レベルなので、もう「LSI」ってよんでる。

## ムーアの法則
- インテル創業者の1人、ゴードン・ムーア
- 1965年「半導体の集積度は18〜24ヶ月で倍増するぞ」っていった人
- ムーアの法則と呼ばれる
- 科学的な根拠はない、感覚による発言
- でも実際そうなった
- 最近は回路のサイズが原子サイズになってこれ以上小さくできない
- でもあれこれ工夫して性能的にはいまだに向上している。



# パソコンの誕生(個人用パソコン)

製造技術の発達で単価もさがり、個人でもコンピュータがモテる時代になってきた。

## 世界初のパソコン、Altair 8800

- １９７４年発売
- 当時最新のCPU(Intel8080)を搭載
- 組み立てキット
- 397ドル
- メモリは256バイト(4キロバイトの拡張メモリボードあり)
- マイクロソフトがこのパソコン用のプログラミング言語 BASICを販売(ぼろ儲け)
- ディスプレイもキーボードもない
- 本体についてるスイッチとLEDランプで簡単な計算結果を表示するだけ
- AltairBUS(S-100バス)を利用して周辺機器を接続して機能拡張できた
- 周辺機器市場が大きく成長
- マイクロソフトも周辺機器市場の参入メーカーのひとつ
- 周辺機器が豊富になってくるとそれらが利用できればAltair意外でもええやんって人増える
- S-100バスが利用できる互換機市場の成長

## Apple Ⅰ
- アップルコンピューター(1976年設立)からApple Ⅰが登場
- 筐体はなく、ケース、キーボード、電源用のトランスなどを自分で用意して組み立てろってもの

## Apple Ⅱ
- パソコン時代きたわ！って事で翌年にAppleⅡを販売
- AppleⅡには世界初の表計算ソフト「VisiCalc」や、豊富な周辺機器の存在があった。
- 優れたソフトや豊富な周辺機器が、普及、拡大に大きく影響した。
- 組み立て式ではなく、キーボード、メモリ、ビデオや音声出力などをオールインワンされた初のパソコン
- テレビやスピーカーへの接続機能も標準搭載
- ホビー用途ではなくビジネス道具としてパソコンが定着するきっかけを作った。
